Pipeline Engineering System for AutoCAD

Automation of pipeline design for oil, gas and water systems

Application Area

  • Gas distribution systems
  • Water supply systems

Purpose

  • Automation of pipeline design
  • Improving design quality
  • Reducing project cost
  • Increasing design speed
  • Minimizing calculation errors
  • Optimizing pipeline routing

Key Features

  • Digital Terrain Model (DTM) creation
  • Pipeline routing with utility detection
  • Surface profile generation
  • Pipeline profile editing
  • 3D earthwork calculations
  • Excel integration
  • Network database support
  • Export to START and CPIPE formats
